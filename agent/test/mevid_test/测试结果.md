## 错误分析

### Q7（cross_camera，✗）
pipeline 追踪结果中 `person_global_2`（浅灰色卫衣）的跨摄像头记录仅覆盖 G329/G328/G421，GE 集合中不存在 G424→G506 的记录。ground truth 标注与 pipeline 追踪结果不一致，属于数据层面问题，agent 无法作答。

### Q8（negative，✗）
"dark hoodie" 描述模糊，G421/G508 均有深色衣物人员，verifier 基于摄像头覆盖判断为 yes。负样本需要更高的外观区分度。

### Q10（negative，✗）
"dark long coat" 与 G421 中的 "dark clothing" 语义高度重叠，verifier 将其判定为匹配。外观描述粒度不足以区分。

## RAG Pipeline 配置说明

| 配置项 | 当前值 | 说明 |
|--------|--------|------|
| 检索层级 | child（track 级） | event 级切换后 IoU 无变化 |
| Parent Projection | 关闭 | 开启后时间窗扩展为 10 分钟，IoU 反而下降 |
| 执行模式 | parallel_fusion | SQL + Hybrid 并行融合 |
| Chroma 集合 | child + global_entity | GE 集合用于跨摄像头 verifier |

**IoU 瓶颈分析**：Mean IoU 0.263 主要受限于 Q5（找到同摄像头不同人的事件，时间窗偏移）和 Q3（ground truth 标注为全程视频 0–299s）。切换到 event-level 检索或启用 parent projection 均不能改善 IoU，当前 child-level 是最优选择。
