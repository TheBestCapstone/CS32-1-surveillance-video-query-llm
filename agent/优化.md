## 4) 针对“进一步提升 context_precision”的可执行方案

以下按收益/风险排序，优先做低风险高收益项。

### A. 先做“无模型变更”的排序提纯（推荐第一优先级）

1. **视频内一致性重排（intra-video consistency boost）**
  - 对 top-K 中属于同一候选视频的 chunk 增加轻量聚合分。
  - 目标：减少“跨视频单条高分噪声”抢占前排。
2. **时间窗口一致性惩罚（temporal incoherence penalty）**
  - 若 query 语义暗含单段事件，惩罚跨度异常离散的候选集合。
3. **高噪声模板词降权**
  - 对 `walking/talking/person/office` 等高频泛化词降低对 rerank 的贡献。

### B. 提升 Tier1 视频粗筛质量（第二优先级）

1. **coarse video filter top-3 -> top-5 动态化**
  - easy/高置信 query 保持 top-3；hard/低置信扩到 top-5。
2. **video discriminator 文本模板增强**
  - 强化“差异性字段”（独特动作、场景结构、角色关系），弱化共性描述。

### C. 调整融合权重（第三优先级）

1. **query-aware RRF 权重**
  - 存在性 + 强结构词：提高 SQL 权重。
  - 关系语义强：提高 hybrid 权重。
2. **partial->mismatch 阈值微调**
  - 针对假阳性高发 query（no-case）提高 mismatch 触发概率。
